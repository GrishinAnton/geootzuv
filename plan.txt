1. Загружается карта
2. При клике на точку в карте мы получаем ее Город, Улицу
3. Всплывает окно, где мы можем оставить отзыв
    3.1 В заголоовку отобраается страна, город улиица
    3.2 Сам отзыв состояит из Блока для отоюражения отзывов и
     блока для ввода отзывов.
    3.3 ПРи нажатии на кнопку добавить отзыв из добавляется в поле
    для отображения отзыво
4. На карте появляется метка, при клике на эту метку открывается окно с отзывами
5. При добавлении еще одного отзыва, эта метка показывает кол-во оставленных отхывов по этому адресу

6. Если кликнуть на одинокую метку, то выводится окно отзыва и добавление
7. Если кликунуть на клластер, то выводит карусель отзывов по этому кластеру
